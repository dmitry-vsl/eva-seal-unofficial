# eva-seal-unofficial

Unofficial pre-built Python wheels for [EVA](https://github.com/microsoft/EVA) — a compiler for homomorphic encryption targeting [Microsoft SEAL](https://github.com/microsoft/SEAL).

The official EVA repository has not been updated since 2021 and does not provide pip packages. This project builds EVA with patches for modern compilers, Python 3.10–3.14, and SEAL 4.1.

## Install

```
pip install eva-seal-unofficial
```

## Supported platforms

| OS | Architecture | Python |
|----|-------------|--------|
| Linux | x86_64 | 3.10–3.14 |
| Linux | aarch64 | 3.10–3.14 |
| macOS | Apple Silicon (arm64) | 3.10–3.14 |
